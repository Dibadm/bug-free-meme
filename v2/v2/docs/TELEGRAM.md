# Telegram Commands

## User Commands

| Command | Description |
|---------|-------------|
| /start | Start the bot and launch Mini App |
| /help | Show help and available commands |
| /profile | View user profile |
| /wallet | View wallet balance and transactions |
| /deposit | Show deposit instructions |
| /withdraw | Show withdrawal info |
| /referral | View referral code and earnings |
| /support | Contact support |
| /leaderboard | View leaderboard |
| /currentrooms | Show current active rooms |
| /mygames | Show user's recent games |
| /history | Show game history |
| /settings | Manage preferences |

## Admin Commands

| Command | Description |
|---------|-------------|
| /admin | Admin dashboard |
| /broadcast | Send announcement to all users |
| /pause | Pause all games |
| /resume | Resume all games |
