# Pages

Route-level page components.

## Purpose

Top-level components rendered for each route.

## Pages

- `HomePage/` - Room listing and dashboard
- `GamePage/` - Live bingo game experience
- `ProfilePage/` - User statistics and settings
- `WalletPage/` - Balance and transactions
- `AdminPage/` - Admin panel

## Conventions

- Each page manages its own data fetching
- Pages compose components from `components/`
- Pages use hooks from `hooks/`
