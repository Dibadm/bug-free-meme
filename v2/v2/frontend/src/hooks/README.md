# Hooks

Custom React hooks.

## Purpose

Reusable stateful logic shared across components.

## Hooks

- `useTelegram.ts` - Telegram WebApp integration
- `useAuth.ts` - Authentication state
- `useWallet.ts` - Wallet balance and operations
- `useGame.ts` - Live game state and websocket
- `useCountdown.ts` - Animated countdown timer
- `useHaptics.ts` - Haptic feedback wrapper
