# Components

Reusable UI components.

## Purpose

Design system components built with Tailwind CSS and Framer Motion.

## Components

- `Button/` - Primary, secondary, ghost variants
- `Card/` - Glass-morphism card container
- `Input/` - Form input with validation states
- `Modal/` - Accessible modal dialog
- `Sheet/` - Bottom sheet for mobile
- `Badge/` - Status and count badges
- `Avatar/` - User avatar with fallback
- `Loading/` - Spinner and skeleton loaders
- `StatCard/` - Animated statistics display
- `Countdown/` - Live countdown timer
- `BottomNav/` - Mobile bottom navigation
- `TopNav/` - Header with actions

## Conventions

- Default export for single-component files
- Props interface named `{ComponentName}Props`
- Variants use `class-variance-authority` or conditional classes
- Animations via `framer-motion`
